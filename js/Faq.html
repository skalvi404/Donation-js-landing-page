What is the Document Object Model (DOM)

The DOM is a tree-like structure that represents HTML documents, allowing JavaScript to interact with and modify web content.

How do you select an element from the DOM?

Use methods like document.getElementById(), document.querySelector(), or document.querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Use setAttribute() for attributes and element.style or classList for styles and classes.
